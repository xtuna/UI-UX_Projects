# Assets

Export images from Figma/FigJam into these folders as you complete each phase. Use PNG for screenshots/exports, keep filenames lowercase-with-hyphens.

- `screenshots-before/` — your original "before" screenshots of the current site/app (already included)
- `wireframes/` — low-fi wireframe exports, e.g. `app-wireframe-01-login.png`, `web-wireframe-01-home.png`
- `mockups/` — high-fidelity mockup exports, same naming convention
- `storyboard/` — storyboard panel images, `panel-01.png` through `panel-06.png`

Committing these as you go (rather than only at the end) gives your GitHub repo a visible history of your process, which is a nice signal for anyone reviewing your portfolio work.
