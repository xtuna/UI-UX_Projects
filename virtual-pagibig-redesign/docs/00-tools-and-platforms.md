# 00 — Tools & Platforms

| Purpose | Tool | Notes |
|---|---|---|
| Wireframes, mockups, prototyping, design system | [Figma](https://figma.com) | Free tier is enough for this whole project |
| Research synthesis (affinity maps, personas, journey maps, storyboard) | [FigJam](https://figma.com/figjam) | Included with your Figma account |
| Interview scheduling / notes | Google Forms + Google Sheets | Free, simple enough for 3-5 lightweight interviews |
| Icon set | [Lucide Icons](https://lucide.dev) or [Phosphor Icons](https://phosphoricons.com) | Free, consistent outline style, Figma plugin available |
| Font | [Inter](https://fonts.google.com/specimen/Inter) | Free Google Font, built into Figma's font list |
| Contrast checking | [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) | Verify AA compliance on every color pair |
| Version control / documentation hosting | [GitHub](https://github.com) | This repo |
| Case study publishing | [Behance](https://behance.net), [Notion](https://notion.so), or a personal site | Pick one, link it from this repo's README |
| Storyboard drawing (optional) | [Storyboard That](https://www.storyboardthat.com) or Figma shapes | Either works fine |

No paid tools are required for this project.
